---

## 🛠️ Tech Stack

This website is built with:

- **Frontend:** React.js, Tailwind CSS, Framer Motion  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB (Mongoose)  
- **Auth:** Firebase Authentication (Google/OTP)  
- **Hosting:** Vercel / Render / Netlify
